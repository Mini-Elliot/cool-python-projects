About EAH

Elliot Alderson’s Hotel (EAH) is a modern hospitality platform designed to simplify hotel operations while delivering a seamless booking experience for guests.

Our Mission

We aim to bridge the gap between traditional hotel management and modern digital solutions. EAH focuses on transparency, efficiency, and user-friendly design — for both hotel staff and guests.

This project demonstrates how technology can streamline reservations, pricing, and customer interactions without overcomplicating the system.

  • ✔ Secure authentication system
  • ✔ Role-based dashboards
  • ✔ Scalable backend architecture
  • ✔ Clean, responsive UI

Technology Behind EAH

Flask

Lightweight backend framework powering routing and logic.

MySQL

Reliable relational database for user and booking data.

Bootstrap

Responsive UI with a consistent design system.

Python
